# Libellus Server

## API entry points
- Status (GET)
- Get Today's Schedule (GET)
- Create Schedule (POST)
- Edit Task (PUT)
- Delete Task (DELETE)
- Delete Many Tasks (DELETE)
- Get Timeline (GET)

## Stack
- Server: Node.js (JS only)
- Database: SQLite (in-memory)
